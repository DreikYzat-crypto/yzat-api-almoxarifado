📦 YZAT Almoxarifado
Sistema profissional de gerenciamento de almoxarifado desenvolvido por Andreik Medeiros (DreikYzat).
O projeto possui versão Web (PWA), API REST em Node.js e aplicativo Android (APK) utilizando Capacitor.
✨ Funcionalidades
🔐 Login com autenticação
👥 Gerenciamento de usuários
🛡️ Níveis de permissão (Administrador, Supervisor e Almoxarife)
📦 Cadastro de produtos
✏️ Edição de produtos
🗑️ Exclusão de produtos
➕ Entrada de estoque
➖ Saída de estoque
📜 Histórico de movimentações
🔍 Pesquisa em tempo real
📊 Dashboard com gráficos
📄 Exportação de relatório em PDF
🌐 Sistema Web (PWA)
📱 Aplicativo Android (APK)
☁️ API hospedada online
🗄️ Banco de dados PostgreSQL
🚀 Deploy automático pelo GitHub
🔄 Integração entre Web, API e Banco de Dados
